# Multi-Step Form

A simple 3-step multi-step form built using HTML, CSS, and TypeScript without any libraries.

## Features

- Collects user details in three steps
- Validation for required fields
- Navigation between steps (Back/Next)
- Data persistence using `localStorage`
- Fully responsive design
- Stepper and progress indicator

## Setup & Usage

1. Clone the repository:

   ```sh
   git clone https://github.com/UmangSachdeva/Multi-step-form
   ```

2. Install the dependencies:

   ```sh
   npm i
   ```

3. Make sure you build the project using
   ```sh
    npm run build
   ```

## Note

Have only used `package.json` to setup typescript, haven't used any libraries for the project
